       README 
===================

 1. Pada folder ini, berisikan dokumen, video, link prototype, dan presentasi terkait Arsitektur Informasi Jasa Titip : Titip!
    (Folder Pada G-Drive : https://drive.google.com/drive/folders/1nvWsHFrw4mV_pwqVIlAVKVhwncQ9IIi5?usp=sharing )

 2. Dokumen terdapat dua jenis file yaitu *.docx dan *.pdf, kedua salinan tersebut berisikan Laporan Perancangan IA Titip!

 3. File Presentasi berjenis file *.pptx berisikan gambaran umum dan poin-poin penting terkait IA Titip!

 4. Link Prototype adalah hyperlink *.txt yang mana jika dibuka akan diarahkan menuju situs Figma yang berisikan prototype IA Titip!

 5. Video berjenis file *.mp4 menjelaskan bagaimana prototype yang telah dirancang berjalan.

 6. Link Video Presentasi adalah hyperlink *.txt yang mana jika dibuka akan diarahkan menuju situs Youtube yang berisikan video presentasi IA Titip!
